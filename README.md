# RBAC Backend Service

Backend-сервис для управления пользователями, ролями и правами доступа (role-based access control).

## Tech stack
- Python
- Flask
- PostgreSQL
- SQLAlchemy
- Flask-Migrate
- JWT
- Docker

## Features
- User authentication
- Role-based access control
- Protected API endpoints
- Health check endpoint

## Project structure
app/
  routes/
  services/
  models/
tests/

## How to run (dev)

1. Create virtualenv
2. Install dependencies
3. Run application
